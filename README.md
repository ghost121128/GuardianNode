# GuardianNode IDS

This is a Mini Intrusion Detection System project.

## Features
- Detects suspicious network activity
- Shows dashboard with charts
- Login system (admin/user)
- Export logs (CSV/PDF)

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run project:
python3 app.py

3. Open:
http://127.0.0.1:5000
